# Other Requirements

- desktop:
    - first 3rd on the top right
    - second 3rd on the left
    - third 3rd on the bottom right
- mobile:
    - first 3rd in top
    - second 3rd in middle
    - third 3rd in bottom

Must use [Vue](https://vuejs.org/) and can use any packer you want: Webpack, Rollup, Parcel, etc.

Feel free to use Flexbox, Grid and/or JavaScript for the mobile/desktop distinction. I'll be using the Chrome dev tools responsive mobile testing for verification.

It doesn't have to be pretty! Just do it as fast as you can and try not to spend too long on it!

# Standing Up the Repo

1. `yarn install`
2. `yarn dev`
